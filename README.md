# RarePizzas
For Laszlo!

# Instructions

- Clone Repository

```
git clone https://github.com/snackman/RarePizzas.git && cd PeaceBonds
```
- Install dependencies

```
npm install
```

- compile contracts

```
npx hardhat compile
```

- run local node

```
npx hardhat node
```

- deploy contracts (in a different terminal)

```
npx hardhat run --network localhost scripts/deploy.js
```
